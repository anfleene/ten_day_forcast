ten_day_forcast by anfleene

This is a super simple heroku based app that will give an ip based ten day forcast

Just screwing around with a couple ruby tools that seems interesting. Weather is totally a solved problem but I thought an IP based ten day forcast might be nice and quick. This is be built on heroku's infrastructure because im cheap and this doesn't need to be fancy. Also it is using rails 3 because I want to experiment with it
